How to Blur Color Image

Summary Use cv2.imread() to read an image file and cv2.cvtColor() to convert the image from one color space (e.g. BGR to RGB). Select the filter (e.g. Gaussian Kernel function cv2.GaussianBlur() or any custom filter cv2.blur()) and apply to the image for smoothing effect. The bigger the kernel size, the blurrier an image will be.
